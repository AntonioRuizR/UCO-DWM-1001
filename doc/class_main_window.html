◆ readData

void MainWindow::readData ( )
slot

Function that receives and process the received data from serial port. Data is received and accumulated until the complete message is formed. Depending on the GUI and device state, each message is procesed differently. To prevent processing failure during GPIO deactivation, while it is in progress, this function only reads and clears serial port buffer.

Depending on the current operating mode, plots distances or position in the corresponding tab.

Defines to which rect of a layout element the size constraints that can be set via \doxylink{class_q_c_p_layout_element_a5dd29a3c8bc88440c97c06b67be7886b}{set\+Minimum\+Size} and \doxylink{class_q_c_p_layout_element_a74eb5280a737ab44833d506db65efd95}{set\+Maximum\+Size} apply. The outer rect (\doxylink{class_q_c_p_layout_element_a2a32a12a6161c9dffbadeb9cc585510c}{outer\+Rect}) includes the margins (e.\+g. in the case of a \doxylink{class_q_c_p_axis_rect}{QCPAxis\+Rect} the axis labels), whereas the inner rect (\doxylink{class_q_c_p_layout_element_a208effccfe2cca4a0eaf9393e60f2dd4}{rect}) does not.

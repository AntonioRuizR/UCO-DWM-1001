Returns the item at the pixel position {\itshape pos}. The item type (a \doxylink{class_q_c_p_abstract_item}{QCPAbstract\+Item} subclass) that shall be taken into consideration can be specified via the template parameter. Items that only consist of single lines (e.\+g. \doxylink{class_q_c_p_item_line}{QCPItem\+Line} or \doxylink{class_q_c_p_item_curve}{QCPItem\+Curve}) have a tolerance band around them, see \doxylink{class_q_custom_plot_a4dc31241d7b09680950e19e5f971ed93}{set\+Selection\+Tolerance}. If multiple items come into consideration, the one closest to {\itshape pos} is returned.

If {\itshape only\+Selectable} is true, only items that are selectable (\doxylink{class_q_c_p_abstract_item_ab36e93c24bc9817f08260e34a38a8759}{QCPAbstract\+Item\+::set\+Selectable}) are considered.

If there is no item at {\itshape pos}, returns {\ttfamily nullptr}.

Plottable\+Type \texorpdfstring{$\ast$}{*} QCustom\+Plot\+::plottable\+At (\begin{DoxyParamCaption}\item[{const QPointF \&}]{pos}{, }\item[{bool}]{only\+Selectable}{ = {\ttfamily false}, }\item[{int \texorpdfstring{$\ast$}{*}}]{data\+Index}{ = {\ttfamily nullptr}}\end{DoxyParamCaption}) const}

Returns the plottable at the pixel position {\itshape pos}. The plottable type (a \doxylink{class_q_c_p_abstract_plottable}{QCPAbstract\+Plottable} subclass) that shall be taken into consideration can be specified via the template parameter.

Plottables that only consist of single lines (like graphs) have a tolerance band around them, see \doxylink{class_q_custom_plot_a4dc31241d7b09680950e19e5f971ed93}{set\+Selection\+Tolerance}. If multiple plottables come into consideration, the one closest to {\itshape pos} is returned.

If {\itshape only\+Selectable} is true, only plottables that are selectable (\doxylink{class_q_c_p_abstract_plottable_a068d169679b7088d28aa3ba5e663dd8c}{QCPAbstract\+Plottable\+::set\+Selectable}) are considered.

if {\itshape data\+Index} is non-\/null, it is set to the index of the plottable\textquotesingle{}s data point that is closest to {\itshape pos}.

If there is no plottable of the specified type at {\itshape pos}, returns {\ttfamily nullptr}.
